# How to use SOURCE

- Install [node](https://nodejs.org/en/)
- Download this (clone the repo)

### Building source

- `npm install`
- `npm start`

### Electron-builder

- Install electron builder
- `env ELECTRON_BUILDER_ALLOW_Uenv ELECTRON_BUILDER_ALLOW_UNRESOLVED_DEPENDENCIES=true yarn dist`
